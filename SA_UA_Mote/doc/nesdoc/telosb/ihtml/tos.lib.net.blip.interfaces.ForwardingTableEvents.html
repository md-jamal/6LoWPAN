Events - Details

defaultRouteAdded

event void defaultRouteAdded()

Signaled when a default route is added to the table; that is, when the stack transitions from not having a default route to having one. This event is not signaled again if the default route changes.

defaultRouteRemoved

event void defaultRouteRemoved()

Signaled when the default route is removed from the routing table. If the route is removed, defaultRouteAdded() be signeled when the route is re-added.
